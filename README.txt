El repositorio cuenta con cuadernos para realizar lo siguiente:

1. dfDulo.R

Homologación de características de las poblaciones utilizadas.Este se encuentra dentro de la carpeta DUFLO, pues se basa en las encuestas de este estudio para migrar la población a las características de interés. Leer "Survey_Set_Eng" y "Cowan 2010" para más detalles del proceso de homologación. Este archivo crea los .csv "dfCowanDufloCI" y "dfCowanDuflo".

2.  descripcion.ipnyb

Simula las poblaciones y crea las gráficas descriptivas de las mismas..

3. modelo.ipnyb

Crea la red neuronal, la entrena y evalúa sus resultados en conjuntos de entrenamiento y validación. Los archivos de la red ya entrenadas se encuentran dentro del repositorio y en modelo.ipnyb se omite el entrenamiento cargando la red directamente. Se anexan también los historiales de error para visualizar el proceso.

4. experimentos.ipnyb

Simula las poblaciones bajo efectos hipotéticos de educación y hábitos sexuales y muestra los resultados.

El análisis de los resultados y comentarios del proceso se encuentran dentro del pdf del trabajo.
